#                           (latest 0.20.0 still declares >=4.56.1, and releases
#                           up to 0.16 actively excluded 5.x via `<5.0.0`).
#       - PYSEC-2025-217    NO fixed version available upstream at all.
#     A blind bump to transformers>=5.0.0rc3 would be an untested major-version
#     combination with surya and is intentionally NOT done. See Task 2 / TODOS.md.
