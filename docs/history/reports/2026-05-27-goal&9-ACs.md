## Acceptance Criteria — Results

### AC-1 — Character-level text selection (browse mode) ✅

- New `PDFModel.get_chars_in_run(page_num, span_id)` returns per-character
  `(text, fitz.Rect)` from `page.get_text("rawdict")`.
- `get_text_selection_snapshot_from_run()` clips start/end runs to the
  hovered character; middle runs render full-width.
- `rawdict` is fetched **once per selection sweep** (passed through as
  optional argument) — no per-run re-parse.
- Asymmetric tolerance handles overlapping baselines (see
  `docs/PITFALLS.md#character-level-run-assignment-fails-for-overlapping-text-lines`).
- Tests: `test_scripts/test_text_selection.py` — same-run partial, cross-run,
  multi-line, copy-roundtrip, rawdict-call-count instrumentation.
- Commits: `58281e8`, `d8628bb`, `bc2ab62` (rawdict reuse).

### AC-2 — Print auto orientation ✅

- `_to_q_page_size()` and the dialog respect source `width > height` per page.
- Mixed-orientation jobs flip per page; explicit override still wins.
- Tests: `test_scripts/test_print_layout.py` — portrait/landscape/mixed.
- Commit: `148ec69`.

### AC-3 — Print auto paper size ✅

- `match_standard_paper_size(w, h, tol=3.0)` matches A0–A6, B4, B5, Letter,
  Legal, Tabloid before falling back to custom `QPageSize`.
- A3 added to dialog combo so users can override explicitly.
- Tie-break (A3 vs nearby custom) verified — see
  `docs/PITFALLS.md#paper-size-matching-tie-break-selects-wrong-size-on-precision-edge`.
- Tests: `test_scripts/test_print_layout.py` — A3 source → A3 QPageSize, A4
  source → named A4, 700×900 → custom.
- Commits: `148ec69`, `57af774`.

### AC-4 — Free drag rotation ✅

- Press-on-handle captures `_rotate_drag_start_angle`; per-frame mouse-move
  emits `RotateObjectRequest(absolute_rotation=...)` (throttled).
- Selection rect + four resize handles all `setTransformOriginPoint(center)`
  + `setRotation(angle)` so the box rotates with the object.
- Click without drag still snaps a 90° step (legacy behavior).
- **Gated to image-like objects** (`image`, `native_image`) — textbox keeps
  legacy 90° step path. See
  `docs/PITFALLS.md#float-rotation-angle-truncated-to-int-on-object-hit-test-retrieval`.
- Tests: `test_scripts/test_object_free_rotation.py`,
  `test_object_free_rotation_gui.py` (incl. textbox negative case).
- Commits: `ccc934f`, `5f913d0`, `bc2ab62` (textbox gate).

### AC-5 — Aspect-ratio resize ✅

- `_mouse_move` resize branch reads `Qt.ShiftModifier`:
  - Plain drag → free-form (arbitrary AR)
  - Shift + drag → AR locked to start-rect ratio
- Default for newly-inserted and native images: free-form.
- Tests: `test_scripts/test_object_resize.py`.
- Commit: `ccc934f`.

### AC-6 — Native PDF images selectability ✅

- `discover_native_image_invocations()` now also enumerates
  `page.get_xobjects()` and recurses into Form XObjects that wrap images.
- Form-space → page-space cm transform reconstructed analytically (see
  `docs/PITFALLS.md#form-space-to-page-space-coordinate-transform-analytical-solution-is-brittle`).
- Verified against `Awareness.pdf`; no regression on `01_報告書.pdf`.
- Tests: `test_scripts/test_native_image_discovery.py`.
- Commit: `8a9c07a`.

### AC-7 — macOS native menu bar ✅

- `_build_macos_menu_bar()` guarded by `sys.platform == "darwin"`.
- Menus: App (About/Preferences/Quit), File (Open/Save/Save As/Print/Close
  Tab), Edit (Undo/Redo/Copy/Paste/Select All), View (Fullscreen/Zoom),
  Window, Help.
- All actions reuse existing `QAction` instances — no duplicates.
- Cmd-W now wired to actual Close Tab handler (fixed in review pass).
- Windows / Linux unaffected.
- Tests: `test_scripts/test_macos_menu.py` (darwin-conditional).
- Commits: `2c53f4b`, `c140f4f`.

### AC-8 — PDF standards conformance ✅

- New `model/pdf_validator.py` exposes
  `check_pdf_conformance(path) -> list[str]`. Checks: PDF header present,
  xref parseable, page tree resolvable, no broken indirect refs.
- New `docs/pdf_compliance.md` lists targeted standards (PDF 1.4+), what
  the validator covers, and how the XREF repair feature acts as the
  conformance safeguard (revised docstring no longer claims same-open-path
  safety — see `bc2ab62`).
- Tests: `test_scripts/test_pdf_compliance.py` — clean fixture returns
  empty list; deliberately-broken fixture reports ≥1 warning.
- Commits: `20e0e42`, `c140f4f`.

### AC-9 — Print spooling speed ✅

- Render path now reuses the cached HiDPI pixmap stream — no double
  rasterisation on print submit.
- 10-page A4 @ 300 DPI completes under the 20 s budget on the reference
  Windows box. Progress dialog updates per page; QThread keeps UI alive.
- Tests: `test_scripts/test_print_speed.py` measures end-to-end spool time
  on a 10-page A4 fixture.
- Commit: `1ba27f0`.
