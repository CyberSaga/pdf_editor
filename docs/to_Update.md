## 問題修復

* 列印完不要改動印表機喜好設定
* 每次編輯完文字方塊會跳到別的位置，把其他位置的文字方塊推走，破壞排版←這個修好幾次了都還沒修好
* 每次編輯完文字，大小也會變（編輯時變得比原本大，完成後變得比原本小），無法保留原樣
* 每次編輯完文字，會固定變成黑色單一字體，無法保留原樣
* 框選文字要以行為單位，非框選「游標經過的文字」而是框選游標經過的「整行」 ← 嘗試修過了沒修好
* 編輯轉向的文字時，文字框裡面的文字也要跟著轉向，而不只是縮窄文字框的寬度 ← 是有了，但編輯時字會放大，超出編輯框就被裁切掉
* 編輯文字時，文字框要透明底，顯示成字將會被儲存的顏色 ← 嘗試修過了只修成平均顏色的底
* 列印要能依頁面方向自動轉向 ← 沒被修到
* 列印要能依來源頁面選擇紙張大小 ← 沒被修到
* 檢視起來還得再更清晰一點（現在文字跟圖片都還是偏模糊）
* 圖片的長寬比會被強制改為選取框的長寬比（1. 剛插入圖片第一次選取時，選取框是正方形，這時候一旦拖動圖片，圖片就會被重製成為正方形；2. 在旋轉圖片的時候，選取框不會跟著旋轉，因此長邊會被縮窄成寬度、寬邊會被拉伸成長度）
* OCR 顯示進度條後到實際開始跑之前有很長的一段時間，CPU 跟 GPU 也都沒有在動，不知道是在等什麼 ← 待查明

* 將所有工作流程都加速（包含：最佳化檔案速度、開啟檔案後換頁、列印送出後卡住電腦的時間）
-> 查明根本成因並修復
