## 解決方案原理

問題在於 CSS 的 `span` 選擇器可能覆蓋內聯樣式中的 `font-family`。現在改為：
- 中文字符（CJK）使用 `cjk` 字體
- 英數字使用 `helv` 字體
- 每個文字段在內聯樣式中明確指定字體，避免被 CSS 覆蓋

這樣可確保中英文混合時，各自使用正確字體，避免英數字間距過大。
